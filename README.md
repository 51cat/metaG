# metaG宏基因组分析工具

## 安装

1. 克隆仓库

   ```shell
   git clone https://github.com/51cat/metaG.git
   ```

2. 安装软件

   ```shell
   cd ./metaG
   bash scripts/build.sh
   ```

3. 测试

   ```shell
   genome_run --help 
   ```
